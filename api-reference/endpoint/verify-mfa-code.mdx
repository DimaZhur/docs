---
title: "Verify MFA Code"
api: "POST https://api.sandbox.altpay.uk/v2/mfa/{type}/sign/{requestId}"
description: "Submits an MFA verification code to retrieve a signToken."
---

### Parameters

- `type` (path) – *(required)* Type of MFA (`sms` or `google`)
- `requestId` (path) – *(required)* ID returned from previous MFA request

### Request Body

Accepts `application/json` with the following field:

- `code` – *(required)* Verification code provided by the user.  
  Max length: 6 characters.

### Authorization

Requires either:
- **Bearer Token** in the `Authorization` header, or  
- **Basic Authentication** (`username:password` in the header).

### Example Request

```bash
curl -X POST "https://api.sandbox.altpay.uk/v2/mfa/sms/sign/abc123" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer <your-token>" \
  -d '{
    "code": "111111"
  }'
```

Example Response

```json
{
  "status": "SUCCESS",
  "data": {
    "signToken": "eyJhbGciOiJSUzI1NiJ9.eyJzaWduSWQiOiI5YjJjZm..."
  }
}
```

On failure, the response may match `ErrorFormResponse`.
