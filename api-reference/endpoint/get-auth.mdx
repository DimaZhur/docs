---
title: "Get Active Tokens"
api: "GET https://api.test.inside.altpay.uk/v2/auth"
description: "Returns a list of active user tokens."
---

### Parameters

- `before` (query) – *(optional)* Filter by creation date before this timestamp.  
- `after` (query) – *(optional)* Filter by creation date after this timestamp.  
- `limit` (query) – *(optional)* Maximum number of tokens to return.  

### Authorization

Requires Bearer Token in `Authorization` header.

### Example Request

```bash
curl -X GET "https://api.test.inside.altpay.uk/v2/auth?limit=10" \
  -H "Authorization: Bearer <your-token>"
```

```json
  [
  {
    "id": "string",
    "userId": "string",
    "token": "string",
    "createdAt": "2024-06-01T12:34:56Z"
  }
]
```
