# WebSocket Client

После выполнения метода `v2/auth/token` вы получите `wsToken`, который нужен для подключения к WebSocket.

Откройте клиент по ссылке ниже и вставьте полученный токен при подключении:

[Перейти к WebSocket клиенту](https://api.sandbox.altpay.uk/ws/client){target=_blank}

---

**Где взять токен?**

Выполните метод `POST /v2/auth/token`, и в ответе будет поле:

```json
{
  "data": {
    "wsToken": "your-websocket-token"
  }
}
