---
title: How can I create JWT token without 2FA?
---

JWT Token creation is only available for sub-users.

**Authorisation:**

- Auth type: Basic  
- Login: sub-user email  
- Password: sub-user password

1. Execute method `POST https://api.sandbox.finsei.com/v2/auth/token`  

2. You will get `signId` in response  

3. Execute:  
   `POST https://api.sandbox.finsei.com/v2/mfa/{type}/sign/{requestId}`  
   - `type` = `google` or `sms`  
   - `requestId` = value from step 2  
   - body:

     ```json
     {
       "code": "111111"
     }
     ```

4. You will get `signToken` value  

5. Execute `POST /v2/auth/token` again with body:

   ```json
   {
     "signToken": "<from previous step>"
   }
   ```

6. Authorisation: Bearer Token with accessToken value from previous step

7. Execute POST /v2/auth with body:

```json
{
  "id": "string",
  "allowIps": ["ip1", "ip2"],
  "withMfa": false,
  "fingerprint": "device-id",
  "expiredAt": 1718700000,
  "app": "FinseiApp"
}
```

8 You'll get signId value

9 Execute again POST /v2/mfa/{type}/sign/{requestId} as in step 3

10 Get signToken

11 Execute final POST /v2/auth with:

```json
{
  "signToken": "<from step 10>"
}
```
